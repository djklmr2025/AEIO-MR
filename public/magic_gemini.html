Instalación en 4 Pasos

1

Descargar

Descarga el archivo ZIP del proyecto y extráelo en tu computadora.

2

Instalar Python

Instala Python 3.8+ y las dependencias necesarias.

3

API Key

Obtén tu API key gratuita de Anthropic y configúrala en el archivo .env

4

Ejecutar

Inicia el servidor y abre tu navegador. ¡Listo para chatear!

# 1. Instalar dependencias pip install flask python-dotenv werkzeug # 2. Configurar API key echo "ANTHROPIC_API_KEY=tu_api_key_aqui" > .env # 3. Ejecutar servidor python server_claude.py # 4. Abrir en navegador # http://localhost:8000
